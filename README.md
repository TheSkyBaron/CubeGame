# CubeGame

Vehicle Editor type game prototype

# How to Use

For demo: https://theskybaron.itch.io/armoria

Project files will be under Assets folder

# Work in progress features

Inventory (not fully done yet)

Spawning vehicles (Framework is in development)

Settings (not created UI yet but it has settings)

Modding (Framework is in development)

# Controls
Esc => Pause menu

Ctrl + a/s/d/w => Rotates Block

1 => Selects Cube

2 => Selects Wedge

Mouse Left Click => Places block

x => Toggles Remove/Place mode

w => Moves player forward

s => Moves player backward

a => Moves player left

d => Moves player right

Mouse Scroll Wheel = Zooms Camera
